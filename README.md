# Trust-Based Landing Page — SenIntel Assignment

A responsive hero section built for a frontend/UI developer assignment. The scenario: a startup launching a secure personal-organization app that helps users store documents, reminders, and checklists in one place.

## 🛠️ Built With
- React (Vite)
- Tailwind CSS

## ✨ Features
- Trust badge highlighting end-to-end encryption
- Clear headline + sub-headline communicating the app's value
- Primary and secondary call-to-action buttons
- Three benefit cards (storage, reminders, checklists)
- Security/trust note near the footer of the section
- Fully responsive — buttons and cards stack on mobile

## 📱 Responsive Behavior
On smaller screens, the two CTA buttons stack vertically instead of sitting side by side, and the three benefit cards collapse into a single column. Font sizes scale down slightly to stay readable without overflowing the viewport.

Claude (Anthropic) was used to help structure the component and draft initial copy. All content was personally reviewed, rewritten in my own words, and tested for responsiveness across screen sizes before submission.
